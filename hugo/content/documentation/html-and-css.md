---
title: "HTML and CSS in Mumble"
---

HTML and CSS can be used for rich text markup, formatting, and styling in Mumble's rich text fields:

- The welcome message of a server (server setting `welcometext`)
- User comments
- Channel descriptions
- Text messages

Mumble uses Qt rich text controls. The subset of allowed and interpreted HTML and CSS declarations can be found in the
[Qt documentation _Supported HTML Subset_](https://doc.qt.io/qt-5/richtext-html-subset.html).
